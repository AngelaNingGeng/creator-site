# AI短视频提示词包

## 关于本产品

专为内容创作者设计的100+专业级AI提示词，覆盖短视频创作的完整流程：选题策划 → 脚本撰写 → 分镜描述 → 标题生成 → 文案润色。

支持 DeepSeek / ChatGPT / Claude / Kimi 等主流AI工具。

## 模块一：选题策划（15个提示词）

### 热点追踪选题
```
你是一个短视频运营专家。请根据当前的社交媒体热点，
为我生成10个[美食/美妆/旅行]领域的短视频选题。
每个选题需要包含：
1. 选题标题
2. 目标受众
3. 预期互动率（高/中/低）
4. 需要的拍摄资源
5. 推荐的BGM风格
```

### 对标账号分析
```
分析抖音账号 @[账号名] 的内容策略：
1. 总结其最受欢迎的5条视频的共同特征
2. 分析其标题结构和钩子技巧
3. 找出其内容矩阵的空白点
4. 建议3个可以超越的方向
```

## 模块二：脚本生成（30个提示词）

### 60秒口播脚本
```
以[主题]为核心，生成一个60秒的口播脚本：
- 前3秒：强钩子（用数字/冲突/好奇心）
- 3-15秒：问题展开
- 15-45秒：解决方案/核心价值
- 45-55秒：情绪高潮
- 55-60秒：行动号召 + 关注引导
风格：[幽默/专业/温暖]
```

### Vlog叙事脚本
```
为一个[地点/活动]的Vlog生成叙事脚本：
- 时间线结构
- 关键转折点（至少3个）
- 情绪曲线设计
- 旁白文案（每段不超过15秒）
- 画面与旁白的对应关系
```

## 模块三：分镜描述（20个提示词）

### 产品展示分镜
```
为一个[产品类型]的展示视频设计分镜头：
1. 开场镜头（0-3秒）：[描述] — 作用：制造好奇心
2. 细节特写（3-8秒）：[描述] — 作用：展示质感
3. 使用场景（8-18秒）：[描述] — 作用：引发共鸣
4. 效果对比（18-28秒）：[描述] — 作用：建立信任
5. 行动号召（28-30秒）：[描述] — 作用：促进转化
```

## 模块四：标题优化（25个提示词）

### 标题生成器
```
为一个关于[主题]的短视频生成10个备选标题，
每个标题需要满足：
- 不超过25个字
- 包含数字或对比
- 激发好奇心
- 适合在信息流中快速抓住注意力
同时为每个标题标注适用场景和预期点击率。
```

## 模块五：文案润色（15个提示词）

### 社交媒体文案改写
```
将以下文案改写为适合抖音发布风格的3个版本：
[原始文案]

版本1：轻松口语化（适合口播）
版本2：悬念引导式（适合图文）
版本3：情绪共鸣式（适合Vlog）

每个版本控制在100字以内。
```
